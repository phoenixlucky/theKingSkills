## 📖 项目介绍

**The King Skills** 是一份面向 **AI Agent（Claude Code / CodeX / Reasonix 等）** 的 Skill 索引清单。

### 🎯 解决什么问题？

> 想给 AI Agent 增加能力，但要一个一个去 GitHub 翻、找安装方法、手动配置——太麻烦。

本项目只做一件事：**标注每个 Skill 的来源地址和安装指令**，让 AI Agent 能**自动识别并一键安装**。

### ✨ 核心优势

| 特性 | 说明 |
|:-----|:------|
| **📦 精选收录** | 84 个 Skill，从 ⭐ 21 万+ 到实用小工具，宁缺毋滥 |
| **🔌 零存储** | 只存索引，不存源文件，永远指向最新版 |
| **🤖 AI 原生** | `skills-index.md` 专为 AI Agent 直接读取而设计 |
| **🌐 可视化网站** | 在线筛选、勾选、一键生成安装指令 |
| **🔄 持续更新** | 追踪社区热门 Skill，定期收录 |

## 🚀 快速开始

### 方式一：🌐 网站批量选择（推荐）

访问 **[theKingSkills 在线工具](https://phoenixlucky.github.io/theKingSkills/)**

```
① 筛选分类 → ② 勾选 Skill → ③ 生成安装指令 → ④ 复制给 AI Agent
```

> ✅ 所见即所得，适合只挑选部分 Skill 的场景。

### 方式二：🤖 AI 直接读取

将 [`skills-index.md`](./skills-index.md) 全文提供给 AI Agent：

```text
请依次阅读并安装以下 Skill 索引文件中的内容。
```

> ✅ 适合需要一次性安装全部或大量 Skill 的场景，一步到位。

## 🔧 扩展指南

想为项目贡献新 Skill？只需 4 步：

```mermaid
flowchart LR
    A[编辑 skills-index.md] --> B[更新 total 字段]
    B --> C[更新一键安装区]
    C --> D[同步 docs/data/skills.json]
    D --> E[提交 PR 🎉]
```

1. **编辑** [`skills-index.md`](./skills-index.md) — 在对应分类下新增条目，包含名称、来源 URL、描述和安装指令
2. **更新** 文件头部 `total` 字段
3. **更新** 底部「一键安装」区的列表
4. **同步** [`docs/data/skills.json`](./docs/data/skills.json) — 确保网站数据一致
